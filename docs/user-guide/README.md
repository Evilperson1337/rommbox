## Overview

RomM LaunchBox Plugin connects LaunchBox to your RomM server so you can browse, import, and optionally install ROMs from inside LaunchBox. It is designed for Windows LaunchBox users who want to manage RomM content without leaving the LaunchBox UI.

> **Note:** This guide assumes you are familiar with LaunchBox basics but does not require any technical knowledge.

## Logs and support info

Logs are stored here:

- `LaunchBox\Plugins\RomM\system\RomM.Plugin.log`

See [Troubleshooting](./troubleshooting.md) for how to gather logs.

## Compatibility

- Windows LaunchBox only.
- Big Box is not supported for the RomM menu items.
