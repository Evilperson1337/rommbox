# Installing the Plugin
Use this guide to install the plugin into LaunchBox.


## Steps
1. Ensure you are on LaunchBox 13.26 or greater
    > If it is an older version, the plugin may not appear in LaunchBox.
2. Download the latest version of the plugin from the *[Releases](https://github.com/Evilperson1337/rommbox/releases)*.
3. Open your LaunchBox installation directory and navigate to the Plugins directory.  
    e.g. *D:\LaunchBox\Plugins*
4. Extract the RomMbox folder into the Plugins directory.  
    e.g. *D:\LaunchBox\Plugins/RomMbox*
5. Open LaunchBox.
